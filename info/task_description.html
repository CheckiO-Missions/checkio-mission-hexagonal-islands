This is another Calculate Islands using a hexagonal grid. Please help the robots again.

  • The size of the hexagonal grid is 12x9. ('A1' to 'L9'),
    • Alphabets represent columns, numbers represent rows.
    • flat topped.
    • shoves even columns down.

You are given a set of coastal hexes for all islands as input value. You have to find all islands that they represent and return an list of that size. (list, tuple or iterable)

Note: The islands doesn't include other islands.

Examples: 

sorted(hexagonal_islands({'C5','E5','F4','F5','H4','H5','I4','I6','J4','J5'})) == [1, 3, 7] 
sorted(hexagonal_islands({'A1','A2','A3','A4','B1','B4','C2','C5','D2', 'D3','D4','D5',
                            'H6','H7','H8','I6','I9','J5','J9','K6','K9','L6','L7','L8'}) == [16, 19]

Input: The coastal hexes of all islands (a set of string).

Output: The sizes of all islands (a iterable (or list, tuple) of integer).

How it is used:

recognizing a area of hexagonal grid.

Preconditions:

  • all(re.fullmatch('[A-L][1-9]', i) for i in input)
