Эта миссия похожа на миссию “Calculate Islands” где вы имеете дело с шестиугольной сеткой. Пожалуйста, помогите роботам снова.

Размер шестиугольной сетки 12x9 (от 'A1' до 'L9').

  • Ее характеристики:
    • буквы алфавита представляют столбцы, а цифры - строки;
    • она имеет плоскую вершину;
    • четные столбцы сдвинуты вниз.

В качестве входного значения вам дается набор прибрежных шестиугольников(гексов) для всех островов. Вы должны найти все острова, которые представляют эти гексы, и вернуть список (кортеж или последовательность) такого размера.

Примечание:

  • Острова не включают другие острова.
  • Если острова нет (ввод == set ()), возвращается [].
  • Также возможен случай, когда нет моря. (В этом случае ваш ответ будет [108].)

Примеры: 

sorted(hexagonal_islands({'C5','E5','F4','F5','H4','H5','I4','I6','J4','J5'})) == [1, 3, 7] 
sorted(hexagonal_islands({'A1','A2','A3','A4','B1','B4','C2','C5','D2', 'D3','D4','D5',
                            'H6','H7','H8','I6','I9','J5','J9','K6','K9','L6','L7','L8'}) == [16, 19]
sorted(hexagonal_islands(set())) == []

Входные данные: Прибрежные гексы всех островов (множество(set) строк(strings)).

Выходные данные: Размеры всех островов (последовательность (список или кортеж) целых чисел).

Как используется:

Для распознавания площади на шестиугольной(гексагональной) сетке.

Предварительные условия:

  • all(re.fullmatch('[A-L][1-9]', i) for i in input)
